# 2026-03-11

## Craft 节点漂浮动画和画布惯性优化

### 完成内容
1. **节点漂浮动画错落有致**：
   - `CraftNode.tsx`：新增 `animIndex` prop，基于索引确定性计算差异化参数（浮动变体 1-3、延迟 0~-6s、周期 5~8s、幅度 12~28px），通过 CSS 自定义属性 `--float-delay`、`--float-duration`、`--float-y` 输出
   - `CraftsPage.scss`：移除原单一 `float` keyframes，定义 3 组运动轨迹 keyframes：`floatY`（纯 Y 轴）、`floatEllipse`（椭圆轨迹）、`floatRotate`（Y+微旋转），通过 `.float-variant-1/2/3` 类名选择
   - `CraftsPage.tsx`：渲染节点时传入 `animIndex={index}`

2. **画布拖拽惯性效果**：
   - 新增 refs：`inertiaRafRef`、`velocityRef`、`lastMousePosRef`、`lastMouseTimeRef`、`viewOffsetRef`
   - `handleMouseDown`：中断惯性 + 初始化速度追踪
   - `handleMouseMove`：记录帧间位移计算瞬时速度
   - `handleMouseUp`：速度 > 1px 时启动 rAF 惯性衰减循环（衰减系数 0.92，阈值 0.5px/frame）
   - `handleWheel`：中断惯性
   - 移除 `canvas-container` 的 CSS transition
